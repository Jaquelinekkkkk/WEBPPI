O que é JavaScrript?

a. A motivação para a criação do JavaScript.

A motivação veio por meio de que os documentos HTML antes do JavaScript eram "estáticos com imagens simples e pouca interatividade". Impossibilitando que os usuários interagissem com a página, assim a navegação era limitada e pouco atrativa.

b. A diferença entre execução no navegador e no servidor.

No navegador, o JavaScript roda ao lado do cliente, tornado mais interativo e dinãmico a manipulação de dados.

A execução pelo servidor ocorre ocorre por um envio de requisição para o servidor, O código JavaScript é executado num ambiente como o Node.js, que não é um navegador. utilizado para gerir dados e operar tarefas em back

c. O conceito de sandbox no navegador.

É um ambiente isolado onde o código poder ser executado, privando o acesso a recursos do sistema e de outros recursos. Limita o acesso a certas funções do navegador, assim garantir a redução de riscos que o navegador e ou o usuário possa sofrer.

d. Objetos globais como document, console e alert.

O document é o arquivo html/página de navegação em que o usuário irá interagir, é o documento atual da tela do navegador.
O console é onde será exibido as informações de mensagens, alertas, >window.history, O >window.location enre outros. window.console ele reprenseta o console JS. O Alert é um alerta que será exibo na janela do navegador.

e. Use id em cada elemento HTML.

f. Utilize console.log() para registrar uma mensagem no console.

g. Utilize alert() para exibir uma saudação ao carregar a página.

h. Exiba a hora atual em um parágrafo com document.getElementById().innerHTML.

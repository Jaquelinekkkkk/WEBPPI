1- Quando o usuário busca uma determinada página html. O navegador manda uma requisição HTTP para o servidor, assim ele retorna com um response. Quando acessado um site, o computador encontra o endereço deste site usando o DNS. Depois, estabelece uma conexão com o servidor usando TCP/IP. O navegador então pede os arquivos do site por HTTP ou HTTPS, depois servidor envia os arquivoss, com isso navegador organiza tudo para mostrar o site na tela do usuario

2- É o padrão para criar páginas web. Html é uma linguagem de marcação e java e javascript é uma linguagem de programao ainda diferentes entre si.

3- Uma regra em css: (P)se refere a paragrafo, (h1) se refere a relevancia de titulos, h1 ate h6, (img) se refere a imagens, então essas propiedades são as regras de html

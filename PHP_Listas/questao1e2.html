Respostas sobre PHP

1. Quais são as tags usadas para definição de código PHP?

As tags usadas para definir código PHP são:

<?php
// Tag padrão
echo "Olá, Mundo!"; // Exemplo de uso

// Tag curta
echo "Usando tag curta!";
?>

 e necessario ser usado o sinal "<..." pq ele meio q engana o arquivo e nao interpreta como os sinais maior q e menor q

Existem duas formas principais de iniciar e finalizar o código PHP:

  • <?php ... ?> (tag padrão)
  • <? ... ?> (tag curta, pode ser desativada em alguns servidores)

1.1. Exemplo de uso de PHP:

O exemplo abaixo exibe a mensagem "Olá, Mundo!" na página:

<?php

 Exemplo de código PHP

echo "Olá, Mundo!";
?>

Ao acessar a página com o código acima, o navegador exibirá:

Olá, Mundo!

2. O PHP é executado no navegador assim como o JavaScript? Explique.

Não. O PHP não é executado diretamente no navegador como o JavaScript. O PHP é uma linguagem de programação do lado do servidor (server-side), o que significa que o código PHP é processado no servidor web antes de ser enviado para o navegador.

Quando o navegador solicita uma página PHP (por exemplo, exemplo.php), o servidor executa o código PHP e envia o resultado (geralmente HTML) para o navegador. O código PHP em si não é visível para o usuário, apenas o HTML gerado.

Em contraste, o JavaScript é executado diretamente no navegador, permitindo interações dinâmicas e manipulação do conteúdo da página em tempo real, sem a necessidade de comunicação com o servidor.

Exemplo de comportamento do PHP e JavaScript:

  • PHP: Processado no servidor e gera HTML para ser exibido no navegador.
  • JavaScript: Executado no navegador e pode modificar o conteúdo da página em tempo real, sem precisar de uma nova requisição ao servidor.
